# TitleCrafters

![demo-app](demo-app.png)

## Requirement

NodeJS: >= 16

## Install

Install packages:

```bash
npm install
```

Build

```bash
npm run build
```

Run development server

```bash
npm run dev
```

Visit dev server on [http://localhost:4173/](http://localhost:4173/)
